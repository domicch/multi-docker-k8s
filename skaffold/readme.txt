Purpose of Skaffold:
In development stage, when a file is updated, the updated file can be directly injected into the running pods to reflect the change

https://skaffold.dev/docs/getting-started 

Step 1:
Install Skaffold with the following command:
brew install skaffold

Step 2:
Create skaffold.yaml file

Step 3:
Run this command:
skaffold dev
